Smart Classroom Timetable Scheduler

Smart Classroom Timetable Scheduler is a web-based application developed using Java, Spring Boot, Thymeleaf, Bootstrap, and MySQL. It helps educational institutions efficiently manage classroom timetables, faculty allocation, and scheduling.

Technologies Used
  • Java
  • Spring Boot
  • Thymeleaf
  • Bootstrap
  • MySQL
Features
  • Admin Login
  • Add Timetable
  • Edit Timetable
  • Delete Timetable
  • Search Timetable
  • Download PDF
  • Export Excel
